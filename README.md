# Parametric 3D Models

> [!NOTE]
> Did you find any of these models useful? Don't forget to give them a rating and post your makes on [Printables](https://www.printables.com/@MOrmesher_4165454/models).

This repo contains the [OpenSCAD](https://openscad.org/) source code for parametric 3D models, many of which also use the excellent [Belfry OpenScad Library](https://github.com/BelfrySCAD/BOSL2/) (BOSL2).

Every model has one or more ready-to-print STL files on [Printables](https://www.printables.com/@MOrmesher_4165454/models), but being _parametric_ means you can also adjust these models to make them suit your needs.

## Customising These models

Editing and exporting OpenSCAD models is easier than you'd think:

1. Download and install [OpenSCAD](https://openscad.org/downloads.html).
1. Clone this repository, or [click here](https://github.com/markormesher/3d-models/archive/refs/heads/main.zip) to download it as a zip.
1. Open the `.scad` file for the model in OpenSCAD.
1. Look for the parameters section, usually towards the top of the file and clearly labelled.
1. Press `F5` to preview your changes, or `F6` to produce a full render (for most models the preview is lower resolution but builds much faster).
1. When you're happy, press `F7` to export and STL file for slicing (or select File → Export → STL).

**Tip:** Start with small changes and preview often. Double check everything before printing, as extreme values may break the geometry.
